## 🚀 Featured Project: Subscription Zombie Hunter
**Problem:** Students often lose track of recurring digital costs—"Subscription Zombies"—from forgotten free trials that drain bank accounts unnoticed.  
**Solution:** I developed a Python tool using the `pandas` library to perform pattern recognition on transaction history, identifying recurring debit cycles (28-31 days) to help peers regain financial control.

## 🛡️ Ambassador Safety Charter
As a Digital Ambassador, I advocate for these core habits:
1. [cite_start]**The 5-Second Rule:** Pause for five seconds to manually inspect a sender's email address or UPI ID before interacting[cite: 429].
2. [cite_start]**The UPI Rule:** Never enter a PIN to receive money; receiving funds is always automatic[cite: 427].
3. [cite_start]**MFA Protocol:** Enable Multi-Factor Authentication (MFA) on all university and personal accounts[cite: 381].
